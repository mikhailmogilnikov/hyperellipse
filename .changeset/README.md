# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish packages.

## What is a changeset?

A changeset is a piece of information about changes made in a branch or commit. It includes a bump type (patch, minor, major) and a summary of the changes.

## When to add a changeset?

Add a changeset when you make changes that should result in a new version of `hyperellipse`.

```bash
bun changeset
```

## Release workflow

```bash
bun version-packages   # bump versions from changesets
bun release            # build and publish hyperellipse
```
